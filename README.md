# 🫂 Silsilah Keluarga

Aplikasi web (PWA) untuk mencatat silsilah keluarga. Dibuat dengan React + Vite.

## Fitur

- ✅ Tambah/Edit/Hapus anggota keluarga
- ✅ Tampilan tree/silsilah
- ✅ Simpan data di IndexedDB (offline)
- ✅ PWA (installable di HP)
- ✅ Desktop & Mobile responsive

## Cara Pakai

### Development
```bash
npm install
npm run dev
```

### Build & Preview
```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- vite-plugin-pwa
- IndexedDB
- CSS
